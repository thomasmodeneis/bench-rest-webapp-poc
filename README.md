# bench-rest-webapp

## Objective

Bench-Rest-WebApp is a POC for having Bench-Rest running with ExpressJS and Firebase on OpenShift RH cloud.


## Usage

* npm install express
* npm install firebase
* npm install bench-rest
* node server.js
